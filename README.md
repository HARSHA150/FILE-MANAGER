# File Manager System

A cross-platform File Management System built using **Java** for Desktop and **Kotlin** for Android. This application provides an intuitive GUI for managing files and directories, including support for creating, deleting, renaming, and viewing files.

## 📌 Features

### Desktop (Java)
- Built using **Java I/O** and **JavaFX/Swing**
- Create, delete, rename files and folders
- GUI-based file explorer
- Supports viewing file content
- Confirmation prompts for irreversible actions
- Error-handling for stability

### Mobile (Android - Kotlin)
- Clean UI with **XML Layouts**
- File/Folder creation and deletion
- Rename and organize files
- Radio buttons & checkboxes for selection
- Works with Android file system permissions

## 🛠 Technologies Used

| Platform | Tech Stack |
|----------|------------|
| Desktop  | Java 17+, JavaFX/Swing, VS Code |
| Mobile   | Kotlin, Android Studio, XML |

## 🚀 Getting Started

### For Desktop:
1. Clone the repository
2. Open the project in **VS Code** or **IntelliJ**
3. Ensure Java JDK 17+ is installed
4. Compile and run the main class

### For Android:
1. Open project in **Android Studio**
2. Build the APK or run on emulator/device
